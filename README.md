## My Arch Linux dotfiles

Setup
-----
```
Bar                     > polybar
Compositor              > picom
File Manager            > gnu/coreutils, fff
Media Player            > ncmpcpp, vlc
Image Viewer            > feh
Program Launcher        > rofi
PDF Viewer              > zathura
Text Editor             > neovim
Terminal Emulator       > termite
Window Manager          > i3-gaps
Browser                 > firefox, w3m
Process Viewer          > htop, gotop
Terminal Fonts          > gohufont
```
